# Slots d'images

Déposez vos images dans ce dossier en gardant **exactement** ces noms de fichiers pour remplacer automatiquement les visuels dans le site :

- `logo-jamaistoutvu.png` → logo principal (header page d'accueil)
- `og-default.png` → image Open Graph par défaut (partage réseaux sociaux)
- `pont-diable.webp` → carte + bannière article Pont du Diable
- `chateau-miglos.webp` → carte + bannière article Château de Miglos
- `cascade-ars.webp` → carte + bannière article Cascade d'Ars
- `aire-jeux-mercus.webp` → carte + bannière article Aire de jeux Mercus-Garrabet
- `fontestorbe.webp` → carte + bannière article Fontaine de Fontestorbe
- `plaine-sports-auzat.webp` → carte + bannière article Plaine des sports d'Auzat
- `louis.webp` → photo fondateur Louis
- `alex.webp` → photo fondateur Alex
- `banner-activites.webp` → bannière page Activités
- `banner-evenements.webp` → bannière page Événements

Conseil : gardez un format proche de 16:9 pour les bannières d'articles et 4:5 pour les portraits fondateurs.
